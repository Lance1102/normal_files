# Polymarket 使用指南

## 概述

Polymarket 是全球最大的預測市場平台，讓用戶可以通過對未來事件進行投注來獲取收益。該平台將新聞、民意調查和專家意見整合成單一數值，反映市場對事件發生機率的觀點。

## 監管狀態

Polymarket 是基於加密貨幣的去中心化預測市場，主要特點：
- 基於區塊鏈技術
- 使用 USDC（美元穩定幣）進行交易
- 用戶間點對點交易，而非對賭場
- 交易對手是其他 Polymarket 用戶

## 如何運作

### 基本概念
1. **買賣股份**：用戶可以購買代表未來事件結果的股份（如「TikTok 會在美國被禁用嗎？」）
2. **價格範圍**：股份價格在 0.00 到 1.00 USDC 之間
3. **完全抵押**：每對事件結果（「YES」+「NO」）由 1.00 USDC 完全抵押
4. **結算**：正確的最終結果股份將結算為 1.00 USDC

### 理解價格
- **價格 = 機率**：價格代表事件發生的當前機率
- 例如：如果 Miami Heat 奪冠的 YES 股份交易價為 18 美分，表示他們獲勝的機率為 18%
- 價格由市場供需決定，Polymarket 不設定價格

### 如何獲利
如果你認為 Miami 的獲勝機率超過 18%，你可以以 18 美分購買「YES」股份。如果 Miami 獲勝，每股「YES」股份價值 1 美元，每股利潤 82 美分。

**重要特點**：
- 可以在任何時間出售股份以鎖定利潤或減少損失
- 沒有「莊家」限制你贏太多
- 價格隨新聞變化而波動

## 預測市場的準確性

研究顯示預測市場通常比專家、民調和評論員更準確。原因：
- 交易者匯集新聞、民調和專家意見
- 經濟激勵確保市場價格反映真實機率
- 更有知識的參與者加入時，市場價格會調整

這使預測市場成為實時事件機率的最佳來源。

詳細 API 文檔：https://docs.polymarket.com/polymarket-learn/FAQ/does-polymarket-have-an-api

## 費用結構

### 交易費用
- **Maker 費用**：提供流動性的用戶通常有返利
- **Taker 費用**：立即成交的訂單收取少量費用
- 具體費率請查看官方文檔

### 獎勵計畫
- **持有獎勵**：持有市場股份可獲得獎勵
- **流動性獎勵**：提供流動性可獲得獎勵
- **Maker 返利計畫**：掛單用戶可參與返利計畫

## 常見問題

### Q: 資金安全嗎？
A: Polymarket 使用區塊鏈技術和智能合約，用戶資金由智能合約保管，通常較安全。詳細信息請查看官方文檔。

### Q: 可以提早賣出嗎？
A: 可以，您可以在任何時間以當前市場價格出售股份。

### Q: 有 API 嗎？
A: 有，Polymarket 提供 API 用於數據訪問和交易集成。
